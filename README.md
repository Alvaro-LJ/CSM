CSM is a R developed toolbox to analyze spatially resolved tissue data.
To see a demo of CSM capabilities please see related
[publication](publicationURL).

## Installation

You can install the development version of CSM from
[GitHub](https://github.com/) with:

``` r
devtools::install_github("Alvaro-LJ/CSM")
```
